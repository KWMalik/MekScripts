Twitter/Brizzly Tools Palette

This script adds extra filtering and embedding features to Twitter Web and Brizzly Web.

Install

  • Chrome: Install from Google
  • FireFox: Install from userscripts.org

Twitter Features

  • Toggle @Mentions and #Hashtags
  • Disable Map Hovercard Pins
  • Filter Location Checkins
    • foursquare (4sq)
    • Gowalla
    • Loopt
  • Filter Personal Streams
    • formspring
    • blip.fm
    • ustream
    • last.fm
    • twitterfeed
    • rhythmbox
    • newzform
    • rss2twitter
    • auto-tweeting
    • chatter.com
    • sharethis
    • alltop
  • Filter Game Statuses
    • mafia wars
    • farmville
    • newtoy games (words/chess with friends)
    • xbox network updates
    • raptr
  • Insert inline Videos and Image Thumbnails
    • Youtube
    • twitvid
    • twitpic
    • yfrog
    • brizzly
    • img.ly
  • Filter Custom Keywords (Filters may also be saved as bookmarklets)

Brizzly Features

  • Toggle @Mentions and #Hashtags
