@meklarian's blip.fm bookmarklet/favelet tools.

To use a bookmarklet, you should right-click and add the bookmark of a link below to your favorites.

If your browser has a 'favorite links' toolbar, place the bookmark there so that you can use it with a single click.

In some browsers, you may also be able to drag the link and drop it directly on the toolbar.

Once you have added a bookmarklet, you may click on it to activate it while viewing a blip.fm page (yours or anyone else). Users of older browsers will need to select them from their favorites menu, instead, while on a blip.fm page.

If you've never used a bookmarklet before, here are some links that can help you learn how to use them.

  • How to use Bookmarklets Video - 5min.com
  • How to use, find, and install Bookmarklets at ehow.com

View Downloadable .mp3 Links

Last Updated on October 12th, 2010

This bookmarklet adds handy links into a blip.fm page to reveal downloadable song links.

Downloadable Blips

In Blip.Fm, these links will appear below the page header. Only songs that have a downloadable .mp3 source will appear in the list shown.
